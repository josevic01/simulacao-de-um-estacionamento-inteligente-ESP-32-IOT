# Sistema Inteligente de Estacionamento com ESP32

Projeto desenvolvido utilizando ESP32 como trabalho final da disciplina de Fundamentos de Sistemas Embarcados, com o objetivo de simular um sistema inteligente de gerenciamento de estacionamento integrando sensores, automação, monitoramento e comunicação via internet.

O sistema realiza a detecção de ocupação de vagas através de sensores, controle de acesso automatizado, monitoramento de luminosidade do ambiente e envio de notificações em tempo real via WhatsApp utilizando API web. O projeto foi desenvolvido com foco em aplicações de Internet das Coisas (IoT), automação e sistemas embarcados.

## Funcionalidades

* Monitoramento de ocupação de vagas
* Controle automático de acesso/portão
* Simulação de cobrança de estacionamento
* Confirmação de pagamento por botão físico
* Envio de notificações via WhatsApp
* Monitoramento automático de luminosidade
* Controle automático de iluminação
* Comunicação Wi-Fi utilizando ESP32

## Componentes utilizados

* ESP32
* Sensor PIR
* Sensor LDR
* LEDs indicadores
* Botão de confirmação
* Comunicação Wi-Fi
* API CallMeBot para integração com WhatsApp

## Tecnologias utilizadas

* Linguagem C/C++
* Arduino Framework
* ESP32
* IoT (Internet das Coisas)
* APIs HTTP
* Automação
* Sistemas Embarcados

## Objetivo do projeto

O projeto foi desenvolvido com foco no aprendizado prático de sistemas embarcados e integração entre hardware e software, explorando conceitos de automação, sensores, comunicação em rede e monitoramento inteligente em tempo real.
